## Project Details

### Project Description (≤ 100 words)
Ur4gent is a marketplace to rent specialized AI agents that help Web3 teams run daily crypto operations on Hedera. Instead of switching across dashboards, teams rent “operators” (Market Analyst, Risk Monitor, Treasury CFO, Payment Operator, Token Creator) and use a unified Command Center to run analysis, schedule payouts, detect anomalies, and review audit trails. Ur4gent also supports OpenClaw bounties with verifiable proofs via Hedera transactions, increasing trust and accountability for community work and treasury workflows.

### Selected Hackathon Track
Main Track: AI & Autonomous Agents (Hedera Operations Automation)

### Tech Stack
- Frontend: Next.js (App Router), React, TypeScript, Tailwind CSS
- Backend: Next.js Route Handlers (API), Node.js
- Database: PostgreSQL (Supabase)
- ORM: Prisma
- Wallet: HashPack via WalletConnect
- Hedera: Hedera SDK (transactions + Hashscan verification links)
- Deployment: Vercel
